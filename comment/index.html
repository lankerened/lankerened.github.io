# 评论

TIP

Aurora主题加入了评论，使用Waline | Walineopen in new window集成评论，Valine是一款基于 Valine 衍生的简洁、安全的评论系统。

# 获取serverURL

请移步到Waline官网，根据他们的教程，获取serverURL

快速上手 | Walineopen in new window

# 配置

在配置文件config.js文件中，

module.exports = {

    themeConfig: {
        comment: {
            //是否显示评论
            showComment: true,
            serverURL: '上一步得到的serverURL地址',
        },
    }
}

# 组件使用

Comment是一个全局组件，目前默认只在关于页面，文章页面以及友情链接页面进行使用，如果在其他页面使用，可以直接<Comment/>
